
# Aplikasi Perhitungan Diskon

# Deskripsi
    "Aplikasi Perhitungan Dsikon". 

Judul Aplikasi: Di bagian atas terdapat teks "Aplikasi Perhitungan Diskon" yang ditampilkan dengan font tebal untuk menunjukkan nama aplikasi.

Field Input Harga Asli:

Terdapat label "Harga Asli" diikuti dengan kotak teks (txtHargaAsli) kosong di sebelah kanannya, yang berfungsi sebagai tempat untuk memasukkan nilai harga awal sebelum diskon.
Dropdown Persentase Diskon:

Terdapat label "Persentase Diskon" dengan dropdown (comboDiskon) di sebelah kanannya yang saat ini menampilkan teks "Pilihan Diskon". Dropdown ini memungkinkan pengguna untuk memilih persentase diskon yang diinginkan.
Field Output Harga Akhir dan Penghematan:

Di bawah dropdown, terdapat dua label "Harga Akhir" dan "Penghematan", masing-masing diikuti dengan kotak teks kosong. Kedua kotak ini akan menampilkan harga akhir setelah diskon diterapkan dan jumlah penghematan yang diperoleh.
Tombol Hitung:

Di bagian bawah antarmuka, terdapat tombol dengan label "Hitung" yang digunakan untuk menghitung harga akhir dan penghematan berdasarkan harga asli dan persentase diskon yang dipilih.

    private void btnHitungActionPerformed(java.awt.event.ActionEvent evt) {                                          
        try {
        // Ambil input harga asli
        double hargaAsli = Double.parseDouble(txtHargaAsli.getText());
        
        // Ambil persentase diskon yang dipilih
        String diskonStr = (String) comboDiskon.getSelectedItem();
        int diskon = Integer.parseInt(diskonStr.replace("%", ""));
        
        // Hitung jumlah penghematan dan harga akhir
        double penghematan = hargaAsli * diskon / 100;
        double hargaAkhir = hargaAsli - penghematan;
        
        // Tampilkan hasil
        txtPenghematan.setText(String.format("%.2f", penghematan));
        txtHargaAkhir.setText(String.format("%.2f", hargaAkhir));
    } catch (NumberFormatException e) {
        // Tampilkan pesan error jika input tidak valid
        javax.swing.JOptionPane.showMessageDialog(this, "Masukkan angka yang valid untuk harga asli.");
    }
    }
